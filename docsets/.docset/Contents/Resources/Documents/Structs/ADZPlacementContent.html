ADZPlacementContent

public struct ADZPlacementContent

Each ADZPlacementDecision one or more ADZPlacementContent instances. Combined, the these represent the creative that should be displayed. A creative may contain a CSS stylesheet and a block of HTML. This would be represented as two contents, one with the type css and one with the type html.
